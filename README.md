# Weather App

A simple weather app that fetches real-time weather data using the Open-Meteo API. Enter a location to get the current temperature and weather conditions.

## Features
- Fetches current weather data based on user-input location.
- Displays temperature and weather conditions.
- Handles errors (e.g., invalid city names).
- Clean and responsive UI design.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/aarti24sh/weather-app

## Usage
1. Open the index.html file in your browser or use a local server.
2. Enter a location in the input field (e.g., "New York").
3. Click the "Get Weather" button to fetch weather information.
4. View the results displayed below the form.
